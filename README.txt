Provide Web Monetization with Coil to WordPress site.

== Description ==

This plugin can register a payment pointer for Web monetization with Coil.

= Features =

* Simple
* New monetization

== Installation ==

From your WordPress dashboard.

1. **Visit** Plugins > Add New
2. **Search** for "WP Connect Coil"
3. **Activate** WP Connect Coil from your Plugins page
4. **Click** on the new settings's submenu item "Coil Setting", and you can register your payment pointer.

= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.6.20 or greater
* MySQL version 5.5 or greater
